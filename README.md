# Dziennik kierowcy

Prosta aplikacja webowa dla kierowcy ciężarówki do codziennego zapisywania danych po zakończeniu pracy.

## Funkcje

- Zapis daty dnia pracy
- Zapis stanu licznika na koniec dnia
- Zapis godziny rozpoczęcia i zakończenia tachografu
- Automatyczne liczenie czasu pracy
- Ręczny wpis czasu jazdy
- Pobieranie lokalizacji GPS na koniec dnia
- Kopiowanie współrzędnych do schowka
- Historia zapisanych wpisów
- Eksport danych do CSV
- Lokalny zapis danych w przeglądarce

## Technologie

- HTML
- CSS
- JavaScript
- localStorage
- Geolocation API
- Clipboard API
